# BSGVerse Portfolio

Welcome to **BSGVerse** — a cyberpunk, neon-themed interactive portfolio for Bishwas Ghimire. This site showcases creative coding, digital artistry, and interactive web experiences, optimized for all devices.

## 🚀 Features

- **AURA.MoodEngine**: Dynamic theme shifter that adapts the site's look and feel based on system time, weather API, or user mood selection. Includes:
  - Auto Dark/Light/Night Sky modes
  - Weather-based backgrounds (sunny, rainy, stormy, etc.)
  - Manual mood selector (Calm, Energetic, Hacker, Dreamy)
  - Real-time CSS theme transitions and UI glow color changes
- **Hero Section**: Animated, rotating lightning sphere with neon glow and cyberpunk effects.
- **Digital Constellations**: Project showcase with interactive cards and unlockable details.
- **BSG Gaming Arena**: Playable Tic-Tac-Toe and Jumper games, fully styled and touch-friendly.
- **Timeline**: Animated digital evolution timeline.
- **Proof-of-Contact**: Crypto puzzle contact form that unlocks after solving a gamified challenge.
- **Sound Effects**: Neon-inspired sound effects with mute/unmute controls.
- **Accessibility**: Supports reduced motion, high contrast, and keyboard navigation.
- **Ultra-Smooth Performance**: Optimized for lag-free scrolling, clicking, and animations on all devices.

## 🎨 Themes & Responsiveness

- **AURA.MoodEngine** provides:
  - Time-based, weather-based, and mood-based themes
  - Smooth transitions and dynamic UI updates
  - Responsive dropdown mood selector in the navbar
- **Fully Responsive**: All layouts adapt for desktop, tablet, and mobile. Font sizes, paddings, and interactive elements scale for readability and usability.
- **Performance Optimized**: Uses hardware-accelerated CSS, reduced animation on mobile, and instant touch feedback.

## 🕹️ Games

- **Tic-Tac-Toe**: Classic game with cyberpunk neon effects, touch/click support, and win/draw detection.
- **BSG Jumper**: Click or tap to jump over obstacles, with score and high score tracking.

## 🧩 Proof-of-Contact

- **Crypto Puzzle Contact Form**: Users must solve a math, pattern, or hash puzzle to unlock the contact form. Includes hints, attempt tracking, and a hidden "🧠 Genius Contacted You" message on submission.

## 🛠️ Setup & Usage

1. **Clone or Download** this repository.
2. Open `index.html` in your browser to view the portfolio locally.
3. For full functionality (contact form, weather API), deploy to a web server and add your API keys as needed.
4. All assets and scripts are included; no build step required.

## 📱 Mobile & Accessibility

- All sections are mobile-optimized and touch-friendly.
- Animations and transitions are reduced or disabled for users with accessibility preferences.
- Keyboard navigation and high-contrast support are built-in.

## 📂 File Structure

- `index.html` — Main portfolio page
- `style.css` — Main styles (neon, cyberpunk, responsive)
- `aura-mood-engine.js` / `aura-mood-engine.css` — Mood engine logic and styles
- `script.js` — Interactive logic (games, unlocks, sound, etc.)
- `puzzle-contact.html` / `puzzle-contact.js` / `puzzle-contact.css` — Proof-of-Contact puzzle form
- `aura-demo.html` — AURA.MoodEngine demo page
