# Genshin Impact-Inspired Portfolio

A visually stunning, interactive portfolio website inspired by the UI and aesthetics of Genshin Impact. Built with HTML, CSS, and JavaScript, this portfolio showcases your skills, projects, and contact information with magical effects and game-like interactivity.

## Features
- **Genshin Impact Theme:** Region backgrounds, elemental icons, vision selector, and more.
- **Animated Background:** Video background with floating magical particles.
- **Interactive Vision Selector:** Instantly change the site’s color theme to match any Genshin element.
- **Character Card About Me:** Styled like a Genshin character profile, with vision badge and splash art.
- **Medium Projects Section:** Smooth, animated project cards with Genshin-style tooltips.
- **Wish Animation:** Genshin-style wish animation on contact form submit.
- **UI Sound Effects:** Subtle fantasy chimes for navigation and actions, with mute toggle.
- **Custom Scrollbar:** Glowing, rounded scrollbar matching the current theme.
- **Region and Vision Icons:** Footer features all Teyvat regions and elements.
- **Responsive Design:** Looks great on desktop and mobile.

## Setup
1. **Clone or Download** this repository.
2. Place your own background video as `background.mp4` in the project root.
3. Open `index.html` in your browser to view the portfolio.

## Customization
- **Name & Avatar:** Edit your name and avatar in `index.html`.
- **Projects:** Update the project cards in the Projects section.
- **Skills:** Edit the skills list and tooltips in the About Me section.
- **Background Video:** Replace `background.mp4` with your own video.
- **Theme Colors:** Easily change theme colors by editing the Vision selector and CSS variables.
