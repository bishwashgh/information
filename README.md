# Tech-Themed Portfolio Website

A modern portfolio website featuring an immersive tech background with floating particle animations and glassmorphism effects.

## Key Features

- **Tech Background Animation**: Dynamic floating particles with smooth animations
- **Glass Effect Components**: Semi-transparent elements with blur effects
- **Project Cards**: Interactive cards with hover transitions
- **Responsive Design**: Mobile-friendly layout with optimized spacing
- **Custom Animations**: Smooth particle movements and transitions

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Layout**: Flexbox and Grid
- **Effects**: Glassmorphism and Particle animations
- **Animations**: CSS keyframe animations
- **UI Components**: Custom-built components

## Project Structure

## Styling Features

- Glass effect components with padding: 1.5rem
- Project cards with consistent spacing: margin-bottom: 1.5rem
- Full-width primary buttons with centered text
- Custom particle animations with staggered delays
- Responsive design with optimized spacing

## Tech Background Details

- Floating particles with smooth animations
- Particle positions optimized for visual appeal
- Animation delays for staggered effects
- Responsive particle positioning

## Usage

1. Open `indexc.html` in a modern web browser
2. View the tech background animation
3. Explore the glass effect components
4. Test responsiveness by resizing the browser window

## Customization

- Adjust particle positions by modifying inline styles
- Modify animation delays for different timing
- Change glass effect padding in CSS
- Adjust project card spacing as needed
